## Project Overview

A console-based C project for managing personal or professional contacts.
Users can add, view, edit, delete, and search contacts by ID.
Uses **file handling** to store data in a text file for future use.
Includes **email validation** logic for correct formatting.
Designed with **beginner-friendly code** and structured functions.
